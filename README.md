# Stackbit Content Packs

Content packs include robust example content for Stackbit themes.

Each content pack includes content and assets.

```
default
├── content
|  ├── data
|  |  └── config.json
|  └── pages
|     ├── blog
|     ├── example.md
|     └── index.md
└── public
   ├── favicon.ico
   ├── images
   |  ├── hero.png
   |  ├── logo.svg
   |  └── water.jpg
   └── videos
      └── hero-video.mp4
```
